# AGRIENGO™ — The Agri Engineer Organic Farms

Welcome to the official repository for the AGRIENGO™ web platform. AGRIENGO™ is a premium organic fertiliser brand dedicated to restoring soil health and empowering gardeners across India through professional-grade soil engineering and agronomist-backed solutions.

## 🌿 Project Overview

AGRIENGO™ is built with a focus on high-performance, rich aesthetics, and seamless user experience. The platform serves as a digital hub for product information, instructional content, and direct conversion pathways to major marketplaces.

### **Core Objectives:**
- **Soil Health Advocacy**: Educating home gardeners on the benefits of organic soil biology.
- **Product Excellence**: Showcasing Grade-A NPK enriched vermicompost.
- **Conversion Focused**: Seamless integration with Amazon, Flipkart, Blinkit, and Swiggy Instamart.
- **Technical Superiority**: Leveraging modern web technologies for sub-second load times and premium animations.

---

## 🛠️ Technology Stack

- **Framework**: [Astro v6.0](https://astro.build/) (Static Site Generation / Hybrid)
- **Styling**: [Tailwind CSS v4.0](https://tailwindcss.com/) (Next-gen CSS engine)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Motion](https://motion.dev/) (High-performance smooth interactions)
- **Runtime**: [Bun](https://bun.sh/) (Fastest JavaScript all-in-one toolkit)
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com/) (Edge-first hosting)
- **SEO**: [Astro-SEO](https://github.com/jonasmerlin/astro-seo) & JSON-LD (Search engine & AI scrapability optimization)

## 🚀 Getting Started

### **Prerequisites**
Ensure you have **Bun** installed on your machine.

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/agriengo-website.git
   ```
2. Install dependencies:
   ```bash
   bun install
   ```
3. Start the development server:
   ```bash
   bun run dev
   ```

### **Building for Production**
To generate a static build optimized for Cloudflare Pages:
```bash
bun run build
```
